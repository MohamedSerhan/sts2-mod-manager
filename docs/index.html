STS2 Mod Manager

A free, open-source Slay the Spire 2 mod manager

Install Slay the Spire 2 mods with one click, manage multiple modpack profiles, and share your setup with friends via short codes. Cross-platform for Windows, macOS, and Linux. No account, no tracking, MIT-licensed.

Download latest release → View on GitHub

What it does

One-click install

Paste a friend's share code and the app downloads every mod from its source (GitHub or Nexus) and applies it.

Modpack profiles

Switch between different mod sets in a single click. Vanilla launch button always available.

Auto-updates

Pulls fresh GitHub releases. Pin a mod to lock it at a known-good version regardless of curator pushes.

Browse Nexus + GitHub

In-app search across both. Free Nexus API key required for Nexus mods.

Auto-backups

Snapshot before every launch. Restore any of the last five with one click.

Audit + diagnostics

Scan installed mods against their sources, see what's outdated, generate a support bundle when something breaks.

Quick start

  • Download the installer for your platform (Windows .msi, macOS .dmg, Linux .deb / .rpm / .AppImage).
  • Run it. The first-launch wizard auto-detects your Slay the Spire 2 install via Steam.
  • Got a code? Paste it on Home and hit Add Pack. The app downloads every mod for you.
  • Hit Launch. Auto-backup runs first.

Source & bug reports
